# zh-vue-cli

## install
```
yarn global add zh-vue-cli
```

## Project setup
```
mkdir workspace

cd workspace

zh-vue-cli create my-project
```

## Project install
```
cd my-project

yarn install
```

### Compiles and minifies for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
